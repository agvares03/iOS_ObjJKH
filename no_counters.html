Показания индивидуальных приборов учета по холодной и горячей воде следует передавать ежемесячно  по 23 число любым удобным для вас способом.

  •  Посредством заполнения соответствующих полей для показаний индивидуальных приборов учета в платежном документе. При этом в расчетах за текущий месяц будут учтены показания, переданные по 20 число включительно.
  • По Интернету через сайт МУП «МРИВЦ» www.mrivc.ru.

При передаче показаний приборов учёта через сайт МУП «МРИВЦ» необходимо указывать соответствующий индивидуальному прибору учета id (идентификатор) счётчика, указанный в ПДУ, и его показание, при этом в расчетах за текущий месяц будут учтены показания, переданные по 23 число включительно.

  • CMC-сообщением на номер +7 903 76 76 007.

Формат передаваемого CMC-сообщения: id счетчика (идентификатор счетчика, указанный в квитанции) 7 знаков и через пробел указать целые показания счетчика. Внимание! На каждый счетчик — свое сообщение. Показания указываются только целыми числами, буквы в тексте сообщения недопустимы! Между id –номером счетчика и показаниями пробел обязателен!
Пример сообщения: 1234567 876.
